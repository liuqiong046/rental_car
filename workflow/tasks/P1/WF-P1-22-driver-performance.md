# WF-P1-22 司机绩效基础统计

| 字段 | 值 |
| --- | --- |
| ID | WF-P1-22 |
| 阶段 | P1 / 阶段一增强 |
| Owner | 蔡佳 |
| 优先级 | P1 |
| 预计体量 | 1-2 天 |

> 说明：owner 负责该任务全栈闭环，不再拆前后端子任务。

## 目标

基于工单完成情况和结算数据形成司机绩效基础金额和基础统计。

## 业务闭环

运营中心和司机可查看与本人或组织相关的基础绩效数据，用于首期经营参考。

## 输入文档

- PRD §6.7、§7.10、§12.3

## 范围内改动

- 司机绩效基础计算。
- 管理端本人绩效基础信息。
- PC 后台司机绩效基础查看。

## 非目标

- 不实现复杂绩效规则、工资结算或高级统计。

## 前置依赖

- `WF-P0-14`
- `WF-P0-18`

## 交付物

- 司机工单数量、完成情况和绩效基础金额。
- 数据范围控制。

## 验收动作

1. 司机只能查看本人绩效基础信息。
2. 运营中心可按数据范围查看司机基础统计。
3. 取消工单不计入完成绩效。

## 完成定义

- 首期司机绩效基础字段可用于导出和后台查看。
